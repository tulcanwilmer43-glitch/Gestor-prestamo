## 3. Nombre del proyecto y detalles

### Nombre del proyecto:
LoanManager Pro

### Descripción:
El proyecto consiste en el desarrollo de un sistema en Python que permite gestionar el préstamo de objetos entre usuarios de manera organizada y eficiente.

El sistema permitirá:
- Registrar usuarios con validación de datos
- Registrar ítems con categorías y precios
- Gestionar préstamos y devoluciones
- Generar notificaciones y reportes
- Controlar el estado de los objetos prestados

Este sistema surge como solución a la desorganización en el control de objetos prestados.

## 6. Especificación de requisitos

### Requisitos funcionales:
- Registrar usuarios con validaciones (nombre, documento, correo)
- Registrar ítems con categoría, precio e ID único
- Crear préstamos asociados a usuarios registrados
- Registrar devoluciones
- Generar certificados de devolución
- Generar facturas de venta luego de 30 días
- Consultar estado general de préstamos
- Validar existencia de usuarios antes de prestar

### Requisitos no funcionales:
- Interfaz amigable por consola
- Seguridad en el manejo de datos
- Uso de archivos planos (CSV)
- Buen rendimiento
- Código estructurado con programación orientada a objetos
